# Performanze — Selfware USA Outbound Strategy

This landing page is structured to be easy to maintain and translate:

- **Design tokens**: `styles.css` defines the Performanze palette + typography (Domine + Inter) as CSS variables under `:root`.
- **Content separation**: all copy/links live in `content.js` as a single `content` object (swap per locale later).
- **Rendering**: `main.js` maps the `content` object to semantic HTML (header/hero/sections/footer).

Case study: RedCoach (6 systems unified, $9K/mo saved). Target: Mid-market transportation & logistics in SE USA.
